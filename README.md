#### BACKEND SETUP

### Requirements

- Python 3.7
- Postgres 11

```
clone the repo
pip install pipenv
pipenv shell
pipenv install
```

### FRONTEND SETUP

#### Requirements

- Node.js and npm

```
cd react_client/
npm install
npm start

```
